===== Data folder for cs-confs-br-data =====

Last major collection occurred in 2025-09, using OpenAlex 
database and complemented by Publish or Perish software on 
databases crossref, also with some manual fixes with Google 
Scholar database.

This folder lists each conference by acronyms.

Category and Subcategory can be retrieved from Google Scholar 
(see GS-categories-2025.txt file).

Conferences from SBC are listed here:
- https://sol.sbc.org.br/index.php/anais/confs

Report from CAPES 2022 Computing committee on the methodology:
- https://www.gov.br/capes/pt-br/centrais-de-conteudo/documentos/avaliacao/09012022_RELATORIOQUALISEVENTOS20172020COMPUTACAO.PDF

Report in XLS:
- https://www.gov.br/capes/pt-br/centrais-de-conteudo/documentos/avaliacao/09012022_CLASSIFICACAODEEVENTOSPARA20172020.xlsx
