===== Data folder for cs-confs-br-data =====

Last major collection occurred in 2025-09, using OpenAlex 
database and complemented by Publish or Perish software on 
databases crossref, also with some manual fixes with Google 
Scholar database.

This folder lists each conference by acronyms.

Category and Subcategory can be retrieved from Google Scholar 
(see GS-categories-2025.txt file).
